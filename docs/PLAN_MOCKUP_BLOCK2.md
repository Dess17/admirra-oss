## Целевой макет (как должно быть)

### 1. Верхний ряд — 3 колонки

| Колонка | Описание |
|---------|----------|
| **Левая: «Статистика по ключевым целям»** | Светло-синий фон. Подзаголовок «За период». Список целей в скруглённых синих прямоугольниках: «Звонки: 14 шт.» + зелёная стрелка «+15.6%». |
| **Центр: «Итого»** | Тёмно-синий фон. Крупная цифра «250 шт.» белым. |
| **Правая: «Разбивка по целям»** | Белый фон. Круговая диаграмма с «250 шт.» в центре. Легенда (синий — Звонки, оранжевый — Quiz, зелёный — Формы). Dropdown «Функциональная». |

### 2. Таблица «Лучшие рекламные кампании»

- Подзаголовок: «По эффективности за период»
- Колонки: Название, Расход, Показы, Клики, CPC, Лиды, CPA
- Валюта: **₽** (рубль)
- Тренд: зелёная стрелка «+15.6%» рядом с каждым значением
- Чередующиеся строки (жёлтый/зелёный фон)

### 3. «Лучшие посты»

- Подзаголовок: «По эффективности за период»
- Горизонтальная галерея карточек с изображениями
- Валюта: **₽**
